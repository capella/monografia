\chapter*{Resumo}

Verificação remota é um método de remotamente verificar a integridade de um dispositivo. Este trabalho visa estudar essa área voltado para dispositivos embarcados. Nele é feito uma revisão bibliográfica dos artigos mais recentes na área. Um artigo específico, denominado \textit{SMART: Secure and Minimal Architecture for (Establishing a Dynamic) Root of Trust} foi utilizado para diversas publicações posteriores, sendo assim, uma referência na área. Esse trabalho também refaz a maior parte dos passos descritos nesse artigo. Além disso, são realizados testes um um FPGA e é criado um servidor com uma infraestrutura completa para a realização da verificação remota via rede. Ataques são realizados o dispositivo nessa estrutura e são identificados com sucesso.
\\

\noindent \textbf{Palavras-chave:} segurança em sistemas embarcados, verificação remota, SMART.

% ---------------------------------------------------------------------------- %
% Abstract
\chapter*{Abstract}

Remote attestation is a method of securely verifying internal state of a remote hardware platform. This work aims to study this area focused on embedded devices. It contains a bibliographical review of the most recent articles in the area. A specific article, called SMART: Secure and Minimal Architecture for (Establishing a Dynamic) Root of Trust has been used for several subsequent publications, being a reference in the area. This work also remakes most of the steps described in this article. In addition, tests are performed on an FPGA and a server with a complete infrastructure is created to test and answer a real device. Attacks are performed on the device and the communication channel, and they were all identified successfully.
\\

\noindent \textbf{Keywords:} embedded systems security, remote attestation, SMART.
